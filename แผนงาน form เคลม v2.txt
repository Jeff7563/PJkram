แผนงาน form V2

1 หน้า index.php 
- เพิ่มช่องให้กรอกเลขไมค์ของรถ ตำแหน่ง : แถวเดียวกับ หมายเลขตัวถัง (d-flex)
- ส่วน แนบรูปภาพปัญหา ให้เพิ่มได้หลายรูป เพราะอันก่อนเพิ่มได้รูปเดียว
- เพิ่มผู้อนุมัติ ของ ซ่อมที่สาขา / ส่งซ่อมที่สนญ. เหมือนกับ เปลี่ยนคันที่มี รหัสพนักงาน (ในส่วนของ รหัสพนักงาน จะทำเป็น drop down เพราะจะดึงจากฐานข้อมูลว่ามีรหัสใครบ้าง พอเลือก รหัส ชื่อกับลายเซ็นต์ ก็จะมาะร้อม) ชื่อ ลายเซ็นต์ แต่อาจจะเอาไว้ ล่าง ปุ่ม เพิ่มรายการ/อัพโหลดรูป
- เพิ่ม แสดงเวลา ว่ารถคันนี้จากวันที่เราขายไป นานแค่ไหนแล้ว ให้เพิ่มช่องใส่วันที่ ขายรถ ไว้ล่างวันที่ส่งเคลม (คำนวณ กี่ปี/กี่เดือน/กี่วัน/กี่นาที) แล้วก็ มีtext สีแดงเอาไว้ระบุวันเวลา แบบ real time

3. edit.php
- ให้ตัดค่าแรงออก ทั้งหมด แล้วเปลี่ยนเป็น เลขที่ job กับ จำนวนเงิน 

4.สิทธิ์ Admin
- ทำ role ขึ้นมา มี admin และ user เพราะว่า โปรแกรมนี้จะเอาไปใช้ในหลายๆ สาขา
- มี Tag Type สร้าง Tag (- ซ่อมที่สาขา / - ส่งซ่อมที่ สนญ. / - เปลี่ยนคัน) ที่มี Tag เพราะว่า หน้าที่การทำงานของ user ต่างกัน บาง user ดูแลแค่ 1 ใน 3 หรือ 2 ใน 3 หน้า index.php ก็จะแสดงแค่ ส่วนงานนั้นๆ ตาม user ที่มี Tag แต่ admin มีทั้ง 3
- หน้า history.php user สามารถ ดูหน้านั้น ได้ แต่จะกด แก้ไขไม่ได้ ดูได้อย่างเดียว ส่วน admin ก็ทำได้หมด
